# Himanshu Shekhar — Industrial Lubricants KAM Portfolio V4

GitHub Pages-compatible static portfolio with the complete V3 design and a browser-based Owner Studio.

## Owner access

- Open `admin.html` or use **Owner login** in the website footer.
- Username: `HSHEKHAR004`
- Change the password in **Advanced**, then export and publish the new `js/content.js`.

## Publishing an edit

1. Log in to Owner Studio.
2. Edit and press **Save changes**.
3. Use **Preview site** to review the browser draft.
4. Press **Export content.js**.
5. In GitHub, replace only `js/content.js` and commit.

See `DEPLOYMENT.md` for the first GitHub upload and document guidance.
